À propos

Parcours & état d’esprit

Depuis 2025, je suis en formation Intégrateur Web chez OpenClassrooms, en visée du titre professionnel RNCP 5 (bac + 2) — Développeur web (2026). Basé à Cergy, je construis des interfaces soignées avec HTML, CSS, JavaScript et React.

Avant la reconversion, j’ai évolué plus de cinq ans en restauration jusqu’au poste de responsable de salle (management d’équipe, formation, encaissements, relation client). En amont, j’ai travaillé en industrie : contrôle qualité, métrologie, assemblage électronique et traitement de surface — un parcours qui renforce ma rigueur et mon sens du détail au quotidien dans le code.

Langues : khmer, langue maternelle · anglais (notions). Permis B. Je recherche une équipe où contribuer à de vrais produits tout en progressant sur la stack front, l’accessibilité, le SEO et les performances.

Formation · Kasa

Kasa

Refonte front-end du site de location entre particuliers Kasa : application React créée avec Vite, navigation multipages via React Router, styles structurés en Sass et respect des maquettes Figma (responsive mobile-first). Données chargées depuis un fichier JSON (annonces) en attendant le back-end. Composants réutilisables : Banner, Card, Collapse (menu déroulant animé, fermé par défaut), Slideshow / galerie (boucle aux extrémités, masquage des contrôles si une seule image, hauteur fixe type object-fit). Gestion des routes : page d’accueil, fiche logement, page 404 et redirection si ID invalide.

Formation · Sophie Bluel

Sophie Bluel

Portfolio d’une architecte d’intérieur pour l’agence ArchiWebos : passage d’une page statique à une application JavaScript connectée à une API REST (Node/Express, Swagger). Galerie alimentée par fetch, filtres par catégorie, page de connexion admin (JWT), mode édition (bandeau, logout, masquage des filtres) et modale pour ajouter / supprimer des travaux avec mise à jour du DOM sans rechargement. Résultat : parcours visiteur fluide + back-office fonctionnel aligné sur la maquette Figma.
